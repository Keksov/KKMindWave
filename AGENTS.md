# Amp Agent Instructions for VMProtect Project

## Pascal Files (.pas)

Use FreePascal compilation binary c:\projects\fpc-trunk\compiler\utils\fpc.exe
Use environment variable FPC_CFG=c:\projects\MindWave\pas\fpc-build\fpc-trunk-x64.cfg for compilation

Target platform x64

When working with Pascal files (.pas and .pp), you **must** analyze and follow the conventions specified in `PASCAL_CONVENTIONS.md`. 

**Before making any changes to Pascal files:**
1. Read `PASCAL_CONVENTIONS.md`
2. Apply naming conventions, memory management patterns, and code style rules
3. Verify all changes conform to the established conventions

This applies to all .pas files in the project
